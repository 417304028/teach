=== 小船过河与关联速度 练习题 ===

--- 知识点 ---

--- 练习题 ---

第1题：小船在静水中的速度大小为4 m/s，水流速度大小为3 m/s，河宽为120 m。求：
(1) 小船的最短渡河时间及此时到达对岸的位置（距出发点的下游距离）；
(2) 小船的最短航程及渡河所需时间。
解析：(1) 最短渡河时间要求船头垂直河岸，渡河时间由河宽除以船在静水中的垂直分速度决定。t_min = d / v船 = 120 / 4 = 30 s。此时船随水漂流，到达对岸时沿河岸方向位移x = v水 * t = 3 * 30 = 90 m。（2）最短航程要求船的实际航线垂直于河岸，即合速度方向垂直河岸。由于v船 > v水，可以满足。船头应斜向上游，与河岸夹角θ满足cosθ = v水/v船 = 3/4，θ = arccos(3/4)。合速度v = v船 * sinθ = 4 * sin(arccos(3/4)) = 4 * √(1-9/16)=4*√7/4=√7 m/s。时间t = d / v = 120 / √7 = (120√7)/7 s ≈ 45.35 s。
答案：（1）最短时间30 s，下游90 m；（2）最短航程120 m，所需时间(120√7)/7 s

第2题：如图所示，人用绳子通过定滑轮拉船，当人在岸上以速度v0匀速向右行走时，求船的速度v船（已知绳子与水平面的夹角为θ）。
解析：将船的速度分解为沿绳方向和垂直于绳方向。船的实际速度v船沿水平方向，绳子收缩的速度等于人的速度v0沿绳方向的分量，但注意：绳子同时绕定滑轮转动，船的速度沿绳方向的分量等于v船 cosθ。由于绳子不可伸长，v0 = v船 cosθ，所以v船 = v0 / cosθ。
答案：v船 = v0 / cosθ

第3题：如图所示，一根轻杆两端分别固定小球A和B，杆可绕中间O点自由转动。当杆与水平方向夹角为θ时，A球的速度大小为vA，方向竖直向下，求此时B球的速度大小vB。
解析：杆两端速度沿杆方向的分量相等，即vA在杆方向的分量等于vB在杆方向的分量。vA与杆夹角为θ（因为vA竖直，杆与水平夹角θ，则vA与杆夹角为90°-θ），所以vA沿杆的分量为vA cos(90°-θ)=vA sinθ。vB方向水平，与杆夹角为θ，沿杆分量为vB cosθ。由vA sinθ = vB cosθ，得vB = vA tanθ。
答案：vB = vA tanθ

第4题：河宽d=200 m，水流速度v水=5 m/s，小船在静水中的速度v船=4 m/s。
(1) 小船渡河的最短时间是多少？此时船头方向如何？
(2) 小船能否垂直河岸渡河？为什么？若不能，小船的最小航程是多少？
解析：(1) 最短时间要求船头垂直河岸，t_min = d / v船 = 200/4 = 50 s。(2) 因为v船 < v水，所以船无法垂直河岸渡河，合速度不可能垂直于河岸。最小航程为船的实际轨迹垂直河岸时，即合速度方向与垂直方向夹角最小。设船头斜向上游与河岸夹角α，则合速度垂直河岸的分量最大时，航程最小。实际上当v船与合速度垂直时，航程最短。最小位移s = d * (v水/v船) = 200 * 5/4 = 250 m。此时船头与上游夹角θ满足sinθ = v船/v水 = 4/5, θ = arcsin(4/5)。渡河时间t = d / (v船 sinθ) = 200 / (4 * 4/5) = 200 / (16/5) = 62.5 s。
答案：（1）最短时间50 s，船头垂直河岸；（2）不能垂直渡河，最小航程250 m，时间62.5 s

第5题：甲乙两船在静水中的速度分别为v1和v2，且v1 > v2。两船从同一渡口出发，都想到达正对岸（即航程最短）。已知水流速度v0 < v1，且v0 < v2。问：甲乙两船能否都垂直过河？若不能，哪一艘船能？若两船船头方向相同，则它们到达对岸的位置关系如何？
解析：要垂直过河，需满足船在静水中的速度大于水流速度，且船头斜向上游。因为v1 > v0, v2 > v0，所以理论上两船都能垂直过河，但实际中船头方向需调整。两船船头方向不同，因为v1和v2不同，所需船头与河岸夹角不同。若船头方向相同，则它们合速度方向不同，到达对岸位置不同，由于v1 > v2，甲船合速度的垂直分量更大，时间短，同时水平分量抵消水流程度不同，具体需计算。但题目问的是“哪一艘船能”以及“位置关系”，实际上只要v船 > v水就能垂直过河。所以两船都能垂直过河。若船头方向相同（比如都与河岸成相同角度），则合速度方向不同，将会偏离正对岸。由于v1 > v2，甲船垂直分速度大，时间短，但水平分速度抵消水流的程度不同，导致漂移不同。但注意：船头方向相同，意味着与河岸夹角相同，则v1和v2在垂直河岸的分量分别为v1 sinθ和v2 sinθ，水平分量分别为v1 cosθ和v2 cosθ（假设斜向上游）。实际合速度水平分量为v1 cosθ - v0（甲）和v2 cosθ - v0（乙），由于v1 > v2，v1 cosθ > v2 cosθ，所以甲的水平分速度更大（可能正或负），但若v0固定，甲更可能顺流漂移。具体位置关系需计算。简化：由于船头方向相同，甲船在垂直方向上速度更大，因此先到对岸，但水平方向上由于v1 cosθ > v2 cosθ，如果v0 < v1 cosθ且v0 < v2 cosθ，那么两船都向上游偏移，甲偏移更多（因为v1 cosθ - v0 > v2 cosθ - v0 >0），所以甲在上游更远处。但题目未给具体数值，所以回答：都能垂直过河；若船头方向相同，甲船到达对岸的位置在上游更远处。
答案：两船都能垂直过河，因为静水速度大于水流速度；若船头方向相同，甲船（速度较大）到达对岸的位置在上游更远处。
